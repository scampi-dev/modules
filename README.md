# scampi modules

> Part of the [scampi](https://scampi.dev) project. Development happens on [Codeberg](https://codeberg.org/scampi-dev/modules) — please file issues and pull requests there. The GitHub mirror exists for discoverability.

Official module library for [scampi](https://scampi.dev) — reusable converge configs for common infrastructure (web servers, containers, APIs, monitoring, etc.).

See the [scampi documentation](https://scampi.dev/docs/) for how to import and use modules in your converge configs.
